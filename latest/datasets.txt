# Graph Algorithms

1. Dijkstra (Shortest Path)
https://github.com/TheAlgorithms/Python/blob/master/graphs/dijkstra.py
Finds shortest paths from a source node in a weighted graph using a greedy strategy
Good for: heap + state consistency bugs

2. Breadth-First Search (BFS)
https://github.com/TheAlgorithms/Python/blob/master/graphs/breadth_first_search.py
Explores nodes level-by-level using a queue
Good for: visited-state bugs, queue duplication

3. Depth-First Search (DFS)
https://github.com/TheAlgorithms/Python/blob/master/graphs/depth_first_search.py
Traverses graph recursively or via stack
Good for: recursion stack bugs

4. Topological Sort (Kahn)
https://github.com/TheAlgorithms/Python/blob/master/graphs/kahns_algorithm_topo.py
Orders DAG nodes based on dependencies
Good for: indegree logic errors

5. Floyd–Warshall
https://github.com/TheAlgorithms/Python/blob/master/dynamic_programming/floyd_warshall.py
Computes shortest paths between all node pairs
Good for: DP state propagation bugs

6. Bellman-Ford
https://github.com/TheAlgorithms/Python/blob/master/graphs/bellman_ford.py
Handles graphs with negative weights
Good for: relaxation logic bugs
